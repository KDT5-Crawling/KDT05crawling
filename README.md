###### AI가 미치는 영향(팀명:Trolling(트롤링))    
* 목차    
- 1. 배경    
- 2. 자동화 및 생산성 향상(이승민)        
- 3. 일자리 감소와 경제적 영향 데이터 보안과 개인정보 우려(김규량)    
- 4. 군사적 AI 활용이 미치는 사회 경제적 영향(김동현)   
- 5. 신뢰성 문제, 윤리적 고민과 규제(AI노래)(명노아)    
- 6. 출처, URL
